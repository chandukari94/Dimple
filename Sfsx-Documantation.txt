VS Code Editor:
---------------

==>Open the vs code editor.

==>open the marketpalce on left side on the monitor.
	-->then type salesforce extension pack
	   -->Install that.
	
==>click ctrl+shift+p then open the command pattle(open the setting icon left side bottom of the monitor,first one command pattle).
==>Then type SFDX:Autherise a Dev hub
	-->then Automatically open login page
	  -->enter the username and password.
==>Then create a scratch org.
	-->open command pattle
	  -->enter create defaultscratch org
	    -->and enter the alias name of the scratch org.
		-->created the scratch org.
		   -->in the command pattle type SFDX:open default org.
		      -->then automatically open scratch org
==>Next,create the project
	-->open the command pattle,type create a project
	   -->then created project.
==>then pull source from scratch org
	-->open command pattel,type SFDX:pull source from scratch org.
	  -->if you create the any objects ,page layots,permissionsets all those come to project in vs code editor.
==>if you develop the code like apexclass,triggers,components,applications etc,.all those push to scratch org
	-->open the command pattle,type push to defult org.
	   -->then automatically push the code vs code editor to scracth org.
		it will display the all code in scrach org developer console.

Github:
------
==>open the marketpalce on left side on the monitor.
	-->type the SCM
	   -->Install that.
==>open the command pattle,Type open github.
	-->Then automaticlally open the login page
	  -->enter the username and password. 
==>open the command pattle,Type GIT:clone
		-->set the repository link.
==>Then open the command pattle,Type Git:add remote
	-->enter remote name
	   -->Enter the repository URL.
==>left side show the git symbel.
	-->then if you changes is happend in your project then automatically show notification.
	  -->commit that changes,Before the commit enter the commit message.

==>then open the command pattle,Type git:push to
	-->enter the remote(just before we create the remote,that is popup bellow,)
	   -->Then automatically store the git hub in your repository.

==>then open the git hub and perticular repository,in that display the version control with commit message.
